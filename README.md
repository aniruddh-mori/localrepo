A simple Currency Converter web application built using HTML, CSS, and JavaScript.  
This project converts one currency into another using live exchange rates.

---

## Features

- Convert currency from one country to another
- Uses live exchange rate API
- Dynamic dropdown for currency selection
- Input validation for amount
- Clean and simple UI

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Currency Exchange Rate API

---

## How to Run the Project

1. Clone or download this repository
2. Open the `index.html` file in any modern web browser
3. Enter the amount and select currencies
4. Click on convert to see the result

---

This project was created to practice:

- JavaScript DOM manipulation
- Fetching data from APIs
- Handling asynchronous operations using async/await
- Real-world frontend development concepts
